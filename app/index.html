Hover for info about the model The model shown above is a graphical representation of our Agent Based simulation. The dots represent agents which reflect the behaviour of people in real life. They have characteristics like "social distancing" and "attraction to centre" which you can change to alter the respective behaviour of the agents. Every agent has a color which is an indication of their current "state" with respect to their infectiousness.The states are as follows Blue-Susceptible Yellow-Non Infectious Red-Infectious Grey-Immune Black- Dead All agents reside in communities,seperated using white borders-which model real life cities and agents can relocate between different communities which can be seen as agents jumping from one community to another. The model corresponds exactly with the graph on the top left.
